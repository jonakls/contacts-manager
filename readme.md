# ContactsManager

ContactsManager is a simple application to manage your contacts efficiently. It allows you to add, edit, delete, and
view contacts with ease.

## Features

- Add new contacts with details like name, phone number, email, and address.
- Edit existing contacts to update their information.
- Delete contacts that are no longer needed.
- View a list of all contacts in a clean and organized manner.
- Search functionality to quickly find contacts by name or other details.

# Requirements

- Java 17 or higher
- Maven or Gradle for dependency management (if applicable)
- A compatible IDE (e.g., IntelliJ IDEA, Eclipse)
- Git for version control
- MySQL or any other database (U need implement database connection by yourself)

## Installation

1. Clone the repository:
    ```
    git clone
   cd contacts-manager
    ```
2. Open the project in your favorite IDE (Use in java 17).
3. Build the project using your IDE's build tools.
4. Run the application.
5. Enjoy managing your contacts!
